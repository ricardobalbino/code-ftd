# Entity

| Attribute | Action        | Description                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| --------- | ------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| name      | update/delete | Technical name of the entity or N:M relation to be modified                                                                                                                                                                                                                                                                                                                                                                                     |
| sheet     | update        | Name of the worksheet where the entity can be found in the template                                                                                                                                                                                                                                                                                                                                                                             |
| key       | update/delete | Technical names of the entity fields which should serve as the lookup keys to determine if a record should be inserted or updated. If multiple keys are defined, those must be separated by the pipe character “|”. Each key must correspond to a column name defined in the column configurations. Only string, whole number and reference entity keys are supported. By leaving out the key attribute, an N:M relationship can be configured. |
| keycolumn | update        | Number of the key column used to determine the break condition when going through the Excel sheet (when multiple keys apply for the column definition, only one of the columns has to be specified for the key column number)                                                                                                                                                                                                                   |
| startrow  | update        | Row number where the list of entity records starts on the template                                                                                                                                                                                                                                                                                                                                                                              |
| action    | delete        | **Optional:** “delete” is the only action which indicates that all entity records shall be deleted. If this attribute is left out, a create/update action is assumed.                                                                                                                                                                                                                                                                           |
| id        | delete        | **Optional:** Required for deletions only if the primary key field of the entity deviates from the schema “<entity name>\_id”                                                                                   
| ignoreselfreferencing | update | **Optional:** For speed up import of self-referencing entities. If true, does not set cache size on 1.
