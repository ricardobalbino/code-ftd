# Troubleshooting

Since the Power Platform is changing a lot, it is possible that also functionalities of the BizApps Core Accelerator related to e.g. Dataverse features are affected. Some common issues are also related to specific workstation setups which must be solved following the given troubleshooting advice.

To support you with the known issues, check the below chapters.

If there is any issue which is not covered by the troubleshooting, please raise a [support ticket](https://platform.avanade.com/support/tickets).
