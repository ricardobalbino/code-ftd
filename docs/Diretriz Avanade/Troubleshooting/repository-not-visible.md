# Azure DevOps Repository not visible

## Problem

You do not see the `Repos` link in the Azure DevOps navigation and have no access to the Git repositories in your project.

## Solution

This is an indication that your Visual Studio license is missing or not properly linked to your account. Make sure that you have a valid `Visual Studio Professional` or higher license which is directly linked to your user account or as an alternate email address.
