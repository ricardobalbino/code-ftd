# Light-Weight Approach

The **BizApps Core Accelerator** can run in **light-weight mode**, which means that the Dataverse solutions can be configured in a no-code or low-code fashion and automatically packaged and deployed to upper environments using the provided pipelines. This approach works for projects of all scales, small to large, and can benefit from the built-in data import and deployment automation capabilities.

As the same structure and tooling is used as for the pro-code mode with backend and frontend code, it is very easy to add coding cabilities to the project and seamlessly transition into this advanced way of building solutions.
