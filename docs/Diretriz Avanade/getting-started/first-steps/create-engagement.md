# Create Engagement

## Create an Engagement

Setting up a new Dataverse engagement has never been easier. The **BizApps Core Accelerator** comes with a comprehensive template that includes everything you need and is fully customizable. Head over to the [**BizApps Platform**](https://platform.avanade.com/engagements) and hit the "Create an Engagement" to get started. The wizard will guide you through the whole process and makes sure your engagement is setup in the most optimal way.

If there are any questions or issues feel free reaching out to the BizApps Platform support via email [dss.platform@avanade.com](mailto:dss.platform@avanade.com) for help.
