# Use Tools

The [Tools](../../tools/index.md) are implicitly utilized or embedded in the BizApps Core Accelerator. There is seamless support in the `bizapps-cli.ps1` to call the tools for typical tasks like data management or deployment automation, which does not require to install and call the tools standalone.

But if necessary, you can perform a local [tool install](../../How-Tos/Others/manage-local-dev-environment.md#install-tools) and then run the tools standalone following the quick start instructions:

* [DPA](../../tools/Deployment-Tool/quick-start-guide.md)
* [EMC](../../tools/Entity-Management-Cockpit/quick-start-guide.md)
