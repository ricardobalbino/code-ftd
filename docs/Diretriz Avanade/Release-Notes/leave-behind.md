# Completely Automated Leave-Behind Process

## Summary

Completely streamlined and automated functionality to cut off all dependencies to Avanade feed and move the relevant code for all tools into the current project repository. After this automated step has been taken, the project teams can independently continue and use and extend the Avanade tools on their own.

This addresses a frequently raised challenge what happens when Avanade leaves a project. With this functionality, it will be even easier for Avanade to leave behind a project because for the tools the existing documentation can be completely reused and handed over to the client team as-is.
