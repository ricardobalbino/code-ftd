# PCF Support

## Summary

Addition and development of PCF components through standard Power Apps Command-Line included in optimized solution flows with extended setup script automation to avoid common mistakes (wrong or missing metadata, suboptimal development structure, missing unit tests etc.).

When a developer wants to add a new PCF component, the DSS CLI takes care of the complete setup and addition to the chosen existing solution. The development can then right start with the local development and even has a skeleton to create end-to-end UI tests in a simple way.
