# Manage Local Development Environment

## Setup Local Development Environment

To quickly set up your local development environment with the right applications, just run this script. See also [Getting Started](../../getting-started/first-steps/installing-tools.md).

1. Run `bizapps-cli.ps1`.
1. Select `SetupLocalDevEnvironment`.

## Install Tools

The main tools EMC and DPA are already accessible and usable via the CLI to manage data and deployment steps but in order to have full and easy access to the latest tool versions, you can use this script which pulls the versions defined in your project from the Avanade feed and gives you the local folder for quick access.

1. Run `bizapps-cli.ps1`.
1. Select `InstallTools`.
1. Select any tool.
1. Go to the tool location (in the local NuGet folder) which is displayed in the script output.
