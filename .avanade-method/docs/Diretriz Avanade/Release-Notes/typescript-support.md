# Full Typescript Support

## Summary
ABC’s frontend core code was completely migrated to Typescript.

## Benefits

| DSS KPI                                               | How the feature helps                                                                                            | Measurements |
|-------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|--------------|
| Increase developer productivity                       | Type definitions help the team implement code faster using the framework                                         |              |
| Decrease automated unit testing effort                | Added seamless Javascript test debugging capabilities in VS Code (same as “Run tests in debug” in Visual Studio) |              |
| Decrease test fix effort with lower number of defects | Type safety to protect the code from being used incorrectly and improve solution quality                         |              |
| Increased maintainability | Type definitions make it easy to pick up extension work as object properties and functions are easily discoverable
