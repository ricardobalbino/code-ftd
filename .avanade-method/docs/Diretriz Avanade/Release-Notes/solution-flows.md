# Significant Improvements and Simplification of Solution Flows

## Summary
Significant improvements to the way how solutions are organized (both functional and technical slicing supported) and how flexibly the solution structure can be changed and extended including the addition of frontend and backend code acceleration.

This supports a wide range of typical project requirements (managed or unmanaged solutions, no-code, low-code or pro-code development).

Furthermore, the deployment automation has been improved and now also considers by default more typical Dataverse deployment tasks such as handling of configuration data.
