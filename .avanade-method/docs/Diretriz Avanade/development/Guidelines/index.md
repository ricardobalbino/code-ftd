# Guidelines

This section describes and defines the process and coding standards for Dynamics / Dataverse Model-Driven Apps and Azure developers.

The following applies to all development and test team members:

-   All members must utilize Azure DevOps, Visual Studio or Visual Studio Code utilizing the source control features.

-   All members must follow the source control layout defined in this document and reflected in the Visual Studio solution stored in Azure DevOps.

-   All members must study and follow the guidelines published in this Wiki as well as in supplemental guidelines such as:
    -   Customizing Guidelines
    -   DevOps Process Automation
