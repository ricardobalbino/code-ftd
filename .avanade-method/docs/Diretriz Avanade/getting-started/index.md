# Getting Started 

We have created this whole section to help you getting started with the **BizApps Core Accelerator** by helping you to setup your environment, understand the project structure, the components comprising the accelerator and start using the included tools and pipelines. In addition, you will be walked through the process of creating your first solution and optionally a plug-in, form script and web resource.

## What is the BizApps Core Accelerator

The BizApps Core Accelerator, BCA for short, is a set of libraries, tools, accelerators and templates that help you be more productive when working with Microsoft Dataverse. It helps standardize the engagement structure to improve quality, enable faster onboarding and increase reusability. The framework provides tooling support, guidance and standardization in the following areas:

- **Customizing and Development Automation**: Fully featured automation tools and Azure DevOps pipelines for solution handling, data management, ...
- **Backend Development**: Modern Engineering Principles, Improved development velocity, automation of manual steps, ...
- **Frontend Development**: Modern web development, Typescript, Webpack & Babel, Early Bound Entities, ...
