# Understanding the Example Code

The engagement you setup through the **BizApps Platform** contains examples to help you get started as quickly as possible. This section guides you through the different examples and helps you understand them.

### Form Scripts

| Name                               | Description                                                                                     |
| ---------------------------------- | ----------------------------------------------------------------------------------------------- |
| account.account.form.controller.ts | Account form scripts which utilizes a standard Crm Sdk implementation approach |
| account.ribbon.controller.ts       | Ribbon script implementation for the account entity |
| contact.contact.form.controller.ts | Contact form script which utilizes a FluentRule approach driven implementation  |
| example.(html/ts)                  | Simple custom html / typescript webresource |
| react-example (folder)             | Contains the boilerplate code for a React based webresource |

### Unit Tests

You can find unit tests in the `tests` folder for each typescript class under the `src` folder.
